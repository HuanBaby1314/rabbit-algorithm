# rabbit-algorithm

author: Jack Huan

# Clone & Install

```shell
# clone
git clone https://github.com/HuanBaby1314/rabbit-algorithm.git
```

```shell
# install by npm => npm i === npm install
npm i
# or install by yarn
# yarn install
```

# test

```shell
# test by npm
npm run test
# or test by yarn
# yarn test
```
